How does Kenbot work?

The backend (behind the scenes) portion of Kenbot relies on a Python algorithm that scrapes information provided in a participation form. A list containing ten variables (each of which is assigned boolean values) is created for each person. As the Kenbot user answers questions about their person, the Python algorithm searches through the database for persons who match the user's answers, and appends these matching persons to the final result list. After ten questions, the properties of the person are unique enough that the final result list contains only one person.

The frontend (what you see) portion of Kenbot uses jQuery and Flask along with the standard HTML and CSS. CSS is used to style the fonts, headers, buttons, and content boxes, along with the background and corner graphics. jQuery is used for the fading animations and hidden information toggles. Flask converts the Python code into a usable format for the web app.
